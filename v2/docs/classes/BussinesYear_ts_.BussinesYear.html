Class BussinesYear

description

ключевой класс для работы формы бронирования, его основной смысл получить из базы данных сведения о машинах и свободных периодов, а затем создать массив дней, где каждому занятому timeSlost будет задан списоок машин, занятых в это время

Также этот класс должен давать возможность получить полный список забронированных timestamp, так и отфильтрованный список по car_id

Кроме того, класс должен давать возможность осуществлять бронирование авто
